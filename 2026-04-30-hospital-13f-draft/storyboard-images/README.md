# Storyboard images

当前模式：草稿模式。

草稿模式不生成图片，只保留后续视觉包所需文件名和提示词。图片提示词以 `deliverable.md` 的“故事板图片提示词”为准。

## 后续视觉包建议生成顺序

1. `character-reference.png`：锁定陈默与妹妹的脸、发型、服装比例。
2. `scene-reference.png`：锁定废弃医院大厅、电梯、13 楼护士站的冷色空间关系。
3. `S01.png`：废弃医院入口。
4. `S02.png`：电梯异常发光。
5. `S03.png`：不存在的 13 楼。
6. `S04.png`：13 楼护士站显形。
7. `S05.png`：陈默跨线前停住。
8. `S06.png`：妹妹坐在护士站。
9. `S07.png`：陈默认出她。
10. `S08.png`：红色弹珠。
11. `S09.png`：弹珠滚到脚边。
12. `S10.png`：门合上前的邀请。

## 规则

- 这里只放静态关键帧，不放视频。
- 文件名必须和 `deliverable.md` 保持一致。
- 红色弹珠是唯一高饱和红色道具，不要漂移。
- 视频合成由外部视频工具完成，Codex 不生成最终 MP4。
